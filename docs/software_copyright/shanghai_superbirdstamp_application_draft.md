## 三、软件概述

`SuperBirdStamp` 是一款面向鸟类摄影图片后期输出场景的跨平台桌面软件与命令行工具，主要用于对鸟类照片进行智能裁切、模板化信息叠加、批量输出和视频导出。软件既可以在图形界面中完成可视化编辑，也支持命令行批量渲染，适用于个人摄影工作流、鸟类拍摄成果整理和社交媒体图文输出等场景。

软件重点解决鸟类照片在“主体居中、比例裁切、元数据横幅、风格模板、批量导出”方面的重复性操作问题，使用户能够用统一模板快速生成适合展示、分享和存档的图片成品。

## 四、开发目的

本软件的开发目标包括：

1. 将鸟类照片裁切、信息排版和样式输出流程整合到一个本地桌面工具中。
2. 通过模板机制减少重复设计工作，提高批量出图效率。
3. 在 GUI 与 CLI 两种模式下统一核心能力，兼顾交互式使用和批处理自动化。
4. 利用元数据、焦点信息和鸟体检测能力，提高裁切中心和版式展示的合理性。
5. 支持中文鸟名和拍摄信息展示，适应中文摄影用户的使用习惯。

## 五、主要功能说明

### 1. 图像打开与批量发现

软件支持打开单张图片或整个目录，并可递归发现目录内的可处理图像文件。输入格式覆盖 JPEG、PNG、TIFF，也支持可选的 HEIF/HEIC/HIF 和 RAW 文件解析。

### 2. 智能裁切与比例控制

软件支持多种输出比例和裁切模式，包括横版、竖版、方形等常见输出规格。用户可在图形界面中调整裁切区域，软件根据图像中心、焦点或鸟体位置进行辅助对齐，从而提高主体展示效果。

### 3. 鸟体检测与主体辅助定位

软件集成基于 `YOLO` 的鸟体检测能力，可在适配条件下自动识别鸟体范围，为裁切中心计算、主体对齐和预览辅助提供依据。这一能力能够减少人工反复试裁的工作量。

### 4. 模板化横幅与文字叠加

软件支持通过 JSON/YAML 模板配置横幅样式、字体、颜色、分隔线、图标和显示字段。用户可以控制鸟名、机身、镜头、快门、光圈、ISO、拍摄时间等字段的展示方式，并保存为自定义模板。

### 5. 元数据解析与规范化

软件能够读取图片 EXIF、XMP 以及相关数据库信息，对拍摄信息进行归一化处理，并为模板渲染构建统一的数据上下文。软件还支持根据文件名规则、命令行参数和元数据内容解析鸟名。

### 6. 实时预览与模板编辑

在图形界面下，用户可以看到裁切结果和模板叠加的即时预览，并实时调整颜色、字体、布局、渐变、边距、显示字段和导出格式。

### 7. 导出图片

软件支持将处理后的图片导出为 JPEG 或 PNG，并支持输出目录、文件名模板、长边限制、图像质量等参数控制，适合批量生成统一风格成品图。

### 8. 视频导出

软件内置视频导出相关流程，结合 `ffmpeg` 工具可以将渲染结果组织为视频帧输出或视频序列处理，扩展了静态图模板在短视频和演示场景中的应用能力。

### 9. 命令行批处理

软件提供命令行接口，可在无图形界面场景下执行批量渲染、图片检查、模板列举和配置初始化。该能力适用于自动化流程、脚本集成和大批量目录处理。

### 10. 模板仓库与配置管理

软件支持内置模板、用户模板和配置文件管理，允许用户保存自己的版式风格，并在后续任务中复用。

## 六、技术实现与系统特点

### 1. 总体架构

`SuperBirdStamp` 采用本地桌面客户端加命令行双入口架构，主要由以下模块构成：

- 图形界面模块：负责模板编辑器、图片列表、预览画布、导出面板和视频导出面板。
- 模板渲染模块：负责模板加载、模板标准化、文字排版、横幅绘制和叠加渲染。
- 图像解码模块：负责多格式图片读取和预处理。
- 元数据模块：负责 EXIF/XMP 读取、字段归一化、鸟名解析和模板上下文生成。
- 智能辅助模块：负责鸟体检测、焦点信息利用和裁切区域计算。
- 批处理模块：负责目录发现、命令行参数解析和批量导出。
- 视频导出模块：负责帧级渲染和 `ffmpeg` 协同。

### 2. 技术栈

结合仓库当前代码与依赖文件，本软件主要技术栈如下：

- 编程语言：Python 3
- 图形界面：PyQt6
- 命令行框架：Typer
- 图像处理：Pillow、numpy、rawpy、pillow-heif
- 模板与配置：PyYAML、JSON
- 鸟体检测：ultralytics、torch、torchvision
- 元数据处理：ExifTool 兼容链路、Pillow EXIF、piexif
- 视频导出：`ffmpeg`
- 打包方式：PyInstaller

### 3. 双入口统一能力

软件通过共享核心处理模块，实现 GUI 与 CLI 共用同一套图片发现、模板渲染、元数据读取和导出逻辑。这样既保证了交互式操作的直观性，也保证了批量处理流程的可复用性和一致性。

### 4. 摄影场景定制能力

软件聚焦鸟类摄影场景，相较一般图片加字工具，具有以下专业特点：

- 以鸟类主体位置和拍摄信息为中心组织裁切与版式；
- 结合焦点、鸟体检测和构图辅助提升画面适配效果；
- 支持中文鸟名和摄影元数据的统一展示；
- 提供适合社交媒体、作品展示和档案整理的固定比例输出。

### 5. 跨平台运行设计

软件在开发与打包层面同时兼顾 Windows 64 位和 macOS，主要体现为：

- 使用跨平台路径与资源定位方式。
- 处理打包后资源目录、配置目录和可执行文件路径差异。
- 对 ExifTool、ffmpeg 等外部依赖的发现和调用进行平台适配。
- 在打包规范中分别维护 macOS 与 Windows 的构建脚本和 spec 文件。

## 七、运行环境说明

### 1. 硬件环境

- 64 位通用个人计算机
- 建议内存 8GB 及以上
- 如启用鸟体检测或大批量渲染，建议更高内存与较新 CPU/GPU 环境

### 2. 软件环境

- Windows 10/11 64 位，或 macOS 近年版本
- Python 3.10 及以上开发环境
- PyQt6 图形库
- Pillow、numpy、PyYAML、rawpy、pillow-heif 等依赖
- `ultralytics`、`torch`、`torchvision`（用于智能检测能力）
- `ffmpeg`（用于视频导出能力）

### 3. 输入与输出对象

- 输入对象：单张照片文件、照片目录、模板文件、配置文件、元数据文件
- 输出对象：叠加模板后的 JPEG/PNG 成品图、模板文件、视频导出中间文件及视频结果

## 八、用户操作流程说明

### 1. 启动软件

用户启动软件后进入编辑器主界面，界面包含图片列表、模板编辑区、裁切设置区、预览区和导出区。

### 2. 打开图片或目录

用户可选择单张图片进行编辑，也可选择目录批量载入待处理图片。系统会自动读取元数据并构建预览所需的上下文信息。

### 3. 选择模板与输出比例

用户可选择内置模板或自定义模板，并设置横版、竖版、方形等输出比例及相关裁切模式。

### 4. 调整裁切与主体位置

用户可在预览区查看裁切结果，并根据图像中心、焦点、鸟体位置或自定义位置对主体进行微调。

### 5. 调整模板样式

用户可修改字体、字号、颜色、背景、渐变、分隔线、显示字段和边距等参数，实时查看叠加效果。

### 6. 导出结果

用户确认效果后，可设置导出目录、图片格式、质量参数和命名规则，批量导出渲染结果。

### 7. 命令行批处理

在需要自动化处理时，用户可以通过命令行执行渲染、检查、模板管理和配置初始化命令，实现无界面批量出图。

## 九、数据处理与文件组织说明

### 1. 元数据来源

软件在渲染过程中会综合读取下列数据：

- 原始图片 EXIF 信息；
- XMP 或辅助 metadata；
- 文件名中的鸟名线索；
- 用户在界面或命令行中指定的覆盖参数；
- 关联数据库或上下文数据。

### 2. 模板数据组织

模板使用 JSON/YAML 形式保存，可配置文字、颜色、位置、比例、导出参数等内容。软件在运行时会先将模板数据标准化，再与图片元数据结合，生成实际渲染上下文。

### 3. 渲染流程

典型渲染过程包括：

1. 发现输入图片；
2. 读取并规范化元数据；
3. 计算输出比例与裁切区域；
4. 结合主体信息与模板参数生成排版结果；
5. 输出为目标格式文件；
6. 如为视频流程，则继续进入帧序列和视频导出步骤。

## 十、原创性与技术特点说明

本软件的原创性主要体现在以下方面：

1. 围绕鸟类摄影图片后期输出构建了从主体裁切、信息排版到批量导出的完整工具链。
2. 结合模板系统、元数据规范化和鸟体检测能力，实现了摄影后期场景下更高自动化程度的版式处理。
3. 统一提供 GUI 实时编辑与 CLI 批量处理能力，使同一核心渲染引擎可应用于交互式和自动化场景。
4. 提供图片导出与视频导出双能力，在同类专业小工具中具有较强的实用扩展性。
5. 对中文鸟名、摄影参数和固定比例展示做了专门优化，适合中文摄影创作者的工作流需求。
